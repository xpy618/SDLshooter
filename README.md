# SDLshooter

一个 2D 竖版射击游戏，使用 SDL2 和 C++17 开发。玩家操控战机，在星空中射击敌人并挑战高分。

## 原作者

Ziyu Shen

本项目代码来自 Ziyu Shen 的 C++ 游戏开发课程练习，仅作学习用途，未做实质性修改。

## 技术栈

- C++17、CMake 3.10+、MSVC
- SDL2、SDL2_image、SDL2_mixer、SDL2_ttf

## 构建

```bash
cmake -B build -S .
cmake --build build
```

构建产物输出到项目根目录。
